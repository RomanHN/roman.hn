Desmos

For each of my Desmos projects, i've included the output from the console command Calc.getState(), just in case Desmos stops hosting the graph for some reason. You can open the Desmos link, or alternatively, you can load the project using Calc.setState() in the console. Simply paste the Calc.getState() text file between the parentheses and press enter. More information at the Desmos API documentation.

This is a printable version of their second short word list, the one with the unique three-character prefixes. "Why don't you capitalize the pronoun 'i'?"

"I" is just like any other personal pronoun, yet it's the only one that's conventionally capitalized. Capitalizing it is a nonsensical convention that i feel the need to resist. Of course, this excludes cases when "i" is capitalized for other reasons - for example, when it's the first word of a sentence.

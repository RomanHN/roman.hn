Desmos

For each of my Desmos projects, i've included the output from the console command Calc.getState(), just in case Desmos stops hosting the graph for some reason. You can open the Desmos link, or alternatively, you can load the project using Calc.setState() in the browser console. Simply paste the Calc.getState() text file between the parentheses and press enter. More information at the Desmos API documentation.
